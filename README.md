Dieses Repository repräsentiert mein modernes Portfolio als **Agentic AI Engineer**. Es demonstriert meine Expertise in der Entwicklung autonomer Agenten-Architekturen, LLM-Integrationen, RAG-Systemen und MLOps. Die UI wurde exklusiv im **KKI-Framework (Liquid Glass Neon Orange)** entwickelt, um High-Performance und futuristisches Design zu vereinen.

## 🚀 Tech Stack
- **Frontend-Architektur:** Semantic HTML5, CSS Variables, CSS Grid, Flexbox
- **Design System:** KKI Liquid Glass (Neon Orange `#FF7A00`, Electric Cyan `#00E5FF`)
- **Effekte:** Neon Glow, Backdrop Blur, Animated Scanlines, Glitch Text Effects
- **Logic:** Vanilla JavaScript, Typewriter Effects
- **Hosting:** GitHub Pages (Automated Deployment)
- **Formulare:** Web3Forms API (Serverless Form Submission)

## 🛠 Local Development
Zum lokalen Testen und Weiterentwickeln genügt ein einfacher Webserver:
```bash
# Repo klonen
git clone https://github.com/KKEEY92/CV_KKEEY.git
cd CV_KKEEY

# Webserver starten
python -m http.server 3000
```
Das Portfolio ist dann unter `http://localhost:3000` verfügbar.

## 🎨 KKI Liquid Glass Framework
Die Kernfarben der Marke **KKI (Kevin Kuck IT)** in diesem Portfolio:

- **Electric Cyan:** `--kk-green: #00E5FF;`
